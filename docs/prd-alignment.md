## 已开发功能的漏洞/风险（本地版）

1. 本地存储明文风险：学习数据与审计日志保存在 `localStorage`，设备被访问时可直接读取/篡改。
2. 隐私口令强度不足：当前是本地轻量哈希（非密码学安全方案），无盐、无抗暴力破解设计。
3. 导出与备份文件外泄风险：导出文件和备份 JSON 默认明文，缺少加密/有效期/访问控制。
4. 审计日志可篡改：审计日志与业务数据同库存本地，导入备份可覆盖历史日志，不具备防抵赖性。
5. 身份校验为本地弱鉴权：手机号/邮箱输入即登录（本地模式），无短信/邮件验证，安全边界弱。
